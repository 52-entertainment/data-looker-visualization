# Data Looker Visualization

A public repository containing reusable JavaScript visualization files for Looker dashboards.

The purpose of this repository is to host custom visualizations and UI enhancements that can be referenced directly from Looker dashboards to improve their look & feel.

## Repository Guidelines

WARNING **This repository is public.**

Only commit files that are intended to be publicly accessible.

### Allowed
- JavaScript visualization files
- CSS assets
- Images, icons, and fonts required by the visualizations
- Documentation (README, examples, etc.)

### Not Allowed
- Secrets, API keys, or credentials
- Configuration files containing sensitive information
- Internal documentation
- Customer or production data
- Any file not intended for public distribution

Before committing, ensure every file is safe to be publicly available.
